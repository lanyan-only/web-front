手风琴特效

使用说明

1.必须引入文件
页面中必须引入JQuery，必须引入fpe.js特效函数库。
注意：JQuery一定要在fpe.js前在head标签对中引入。
代码：
<script src="jquery.js"></script>
<script type="text/javascript" src="fpe.js"></script>
2.页面布局
布局：需有一个父容器，父容器中为内容列表，如：我这里列表内容为a标签。
建议：列表最小不要少于3个最大不要超过10个
布局示例： 
	<div class="uek_box">
		<a href="#"><img src="images/1.gif" /></a>
		<a href="#"><img src="images/1.gif" /></a>
		<a href="#"><img src="images/1.gif" /></a>
		<a href="#"><img src="images/1.gif" /></a>
	</div>
3.函数使用说明
函数名：accordion
$(obj).accordion(lists)
obj：为父容器
lists：为内容列表
示例：
	<script>
 		$(function(){
 			$('.uek_box').accordion($('.uek_box a'));
 		}) 
	</script>
