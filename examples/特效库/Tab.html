使用说明

1.必须引入文件
页面中必须引入JQuery，必须引入fpe.js特效函数库。
注意：JQuery一定要在fpe.js前在head标签对中引入。
代码：
<script src="jquery.js"></script>
<script type="text/javascript" src="fpe.js"></script>
2.页面布局
布局：需有一个父容器，父容器中分为两部分，第一部分为 链接列表，第二部分为 内容列表。
布局示例： 
	<div class="box">
            <div class="links">
                <a href="#">新品推荐</a>
                <a href="#">热销产品</a>
                <a href="#">销量榜单</a>
            </div>
            <ul class="lists">
                <li><img src="images/ls1.jpg" /></li>
                <li><img src="images/ls2.jpg" /></li>
                <li><img src="images/ls3.jpg" /></li>
            </ul>
	</div>
3.函数使用说明
函数名：tab
obj.tab(links,lists,[bgcolor],[ftcolor]);
参数obj：父容器
必选参数links：选择点击选项列表。
必选参数lists：选择内容列表。
可选参数bgcolor：改变链接背景颜色。
可选参数ftcolor：改变链接文字颜色。
示例：
	<script>
		$(function(){  
            $(".box").tab($(".links a"),$(".lists li"))
            //$('.box') 父容器     $(".links a") 链接列表   $(".lists li")内容列表
		}) 
	</script>
