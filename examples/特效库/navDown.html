下拉导航特效

效果展示

  • 首页
  • 新品上市
    今日新品 本月新品 上月新品
  • 热销产品
    本日热销 本月热销
  • 特惠产品
    今日特惠 本月特惠
∧

使用说明

1.必须引入文件
页面中必须引入JQuery，必须引入fpe.js特效函数库。
注意：JQuery一定要在fpe.js前在head标签对中引入。
代码：
<script src="jquery.js"></script>
<script type="text/javascript" src="fpe.js"></script>
2.页面布局
布局：需有一个父容器，父容器中为内容列表，每个分类放入一个li中，每个li中包含下拉列表并且放入一个容器中。
布局示例： 
	<ul class="nav">
		<li>
			<a href="#">首页</a>
		</li>
		<li>
			<a href="#">新品上市</a>
			<div>
				<a href="#">今日新品</a>
				<a href="#">本月新品</a>
				<a href="#">上月新品</a>
			</div>
		</li>
		<li>
			<a href="#">热销产品</a>
			<div>
				<a href="#">本日热销</a>
				<a href="#">本月热销</a>
			</div>
		</li>
		<li>
			<a href="#">特惠产品</a>
			<div>
				<a href="#">今日特惠</a>
				<a href="#">本月特惠</a>
			</div>
		</li>
	</ul>
3.函数使用说明
函数名：navDown
obj.navDown(lists,dlists);
参数obj：父容器
参数lists：选择内容列表
参数dlists：选择下拉内容列表
示例：
	<script>
		$(function(){
	 	$('.nav').navDown($(".nav li"),$(".nav li div"));
	 	//$('.nav') 父容器     $(".nav li") 内容列表     $(".nav li div") 下拉列表   
		}) 
	</script>
